Crime Tracker

Project description

This website was developed to help the citizens to report an FIR against crime from anywhere in India. So that police can take action against crime immediately.

Here every citizen needs to register themselves by ADHAAR CARD so that no fraud complains are registered and if such things happens then that person will be identified easily and action can be taken against fraud complains. Anyone can report an FIR. That FIR directly goes to headquarter and that officer will reassign that FIR to the local police station.

And after that police will update that FIR and person who has reported the FIR can see the updates and if police is not doing anything he/she can write message to headquarter.

Users:

Indian citizens: Any Indian citizen can register by using ADHAAR card number.

Police officers: Local police officers.

Head quarter: IPS officers to assign case to police station.

My features:

  1. Registration/Login
  2. Reporting crime
  3. View investigation report
  4. Missing person report
  5. View missing person report
  6. Missing vehicle report
